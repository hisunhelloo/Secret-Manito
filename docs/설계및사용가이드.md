# 🎁 비밀 마니또 - 설계 및 사용 가이드

**비밀 마니또(Secret Manito)**는 서버 구축 없이 가볍게 즐길 수 있는 마니또 매칭 웹 서비스입니다.

---

## 🛠 주요 기능

### 1. 방 관리 (Administrator)
- **방 만들기**: 본인의 이름과 모임(방) 이름을 입력하여 새로운 방을 생성합니다.
- **멤버 관리**: 참여할 친구들의 이름을 명단에 추가합니다. (잘못 추가한 경우 삭제 가능)
- **마니또 배정**: 멤버가 모두 모이면 알고리즘을 통해 마니또를 배정합니다.
  - *주의: 배정이 시작되면 멤버를 수정할 수 없습니다.*
- **초대 링크 공유**: 각 멤버별로 생성된 **전용 링크**를 복사하여 카카오톡이나 DM으로 전달합니다.
  - *보안: 방장이라도 누가 누구의 마니또인지 알 수 없습니다.*

### 2. 결과 확인 (Member)
- **비밀 확인**: 전달받은 링크로 접속하면 카드 뒤집기 애니메이션과 함께 내 마니또를 확인할 수 있습니다.
- **보안 접속**: URL에 포함된 고유 토큰을 통해서만 접근 가능하므로, 남의 결과를 볼 수 없습니다.

---

## 🎲 마니또 배정 로직
- **원형 순환 매칭 (Circular Logic)**:
  - `A -> B -> C -> A` 형태로 꼬리에 꼬리를 물도록 배정합니다.
  - 누구 한 명이라도 배정받지 못하거나(Null), 자기 자신을 뽑는(Self) 경우가 발생하지 않도록 설계되었습니다.

---

## 🚀 사용 방법 (Quick Start)
1. **[방 만들기]** 메뉴에서 모임 이름과 본인 이름을 입력하세요.
2. **[+ 멤버 추가]** 버튼으로 친구들의 이름을 모두 등록하세요.
3. 명단이 완성되면 하단의 **[배정 시작]** 버튼을 누르세요.
4. 각 멤버 이름 옆의 **[링크 복사]** 버튼을 눌러 해당 멤버에게 개인 메시지로 보내주세요.
5. 친구들은 링크를 열어 자신의 마니또를 확인하면 끝! 🎁
