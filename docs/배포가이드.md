# 🚀 배포 가이드 (Deployment)

친구들이 언제나 접속할 수 있도록 인터넷상에 웹사이트를 배포하는 방법입니다.
**Firebase Hosting**을 사용하면 무료로 쉽고 빠르게 배포할 수 있습니다.

---

## 1. Firebase 도구 설치
한 번도 설치한 적 없다면, `firebase-tools`를 설치해야 합니다.
(이 프로젝트에는 로컬 의존성으로 설치되어 있어 `npx`로 실행 가능합니다.)

```bash
npm install -g firebase-tools
```

## 2. 로그인
Firebase에 배포하려면 로그인이 필요합니다.

```bash
# 기본 로그인
firebase login

# 오류 발생 시 (웹브라우저가 안 뜨는 경우)
npx firebase login --no-localhost
```

## 3. 배포하기 (Deploy)
**빌드(Build)**와 **업로드(Deploy)**를 한 번에 처리하도록 스크립트가 준비되어 있습니다.

```bash
npm run deploy
```

위 명령어를 실행하면:
1. `npm run build`: 소스 코드를 최적화하여 `dist` 폴더에 생성합니다.
2. `firebase deploy`: 생성된 `dist` 폴더를 Firebase 서버로 전송합니다.

## 4. 완료
배포가 끝나면 터미널에 **Hosting URL**이 출력됩니다.
예: `https://secret-manito.web.app`

이 주소를 친구들에게 공유하면 됩니다! 🎉
